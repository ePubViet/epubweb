Welcome to TQuotes Help File

Getting Started

Quick Start:

Thanks for installing Tquotes Module
On the left side of the screen (Details) select a position to display module
Select Status : Published
Scroll dowm to Menu Assignment
Select : On all Pages
Goto top of screen and select : Save & Close

This will display The sample File (tquotes.txt) with the module Defaults on all your pages.
Random, No Quote Marks , use the "|" charactor as the seperator and use the Template styling.
All the remaining parameters will be overridden.

More Options:

You can add as many of your own textfiles as you want. Just upload it to your server into the /module/mod_tquotes/mod_tquotes directory and it will appear in the list of files to choose from. A .txt file extension is required.

File Requirements:

Each line in the file will be a seperate quote. If you use a seperator between the quote and the author they will appear seperately. Do not have any blank lines in the file.
Example file:
This line is a quote. | this is the author. //use any seperator you like
This line is a statement to display and does not have an author.
This line can refer to an image and it will display.

You can have as many lines as you like depending on your rotation choise.

Randon : will choose a quote from the an unlimited number of lines.

Daily: will rotate quotes each day. The quote chosen will depend on the day of the year up to 366 days. If less then 366 lines the quotes will start at Line one and repeat again. // line 367 will not show and 366 will show once every leap year.

Weekly: Will rotate each week for the 52 weeks of the year and repeat again. // Lines limited to 52. Line 53 and beyond will not display.

Monthly: Will rotate each month for 12 months and repeat again. Lines limited to 12. More will not display.

Every Minute: Will rotate every minute( 60 ) . Lines limited to 60. More will not display.

Every Hour: Will rotate every hour (24). Lines limited to 24. More will not display.
